# ft_transcendance
Ce projet est une application web complète autour du jeu Pong, combinant front-end, back-end et gestion temps réel. J'ai contribué sur la gestion de projet, la création du jeu, du système de tournois et d'un adversaire IA, tout cela côté front-end et back-end.
